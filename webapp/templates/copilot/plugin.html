Setup Instructions

1.
Enable HttpService — In Roblox Studio, go to Game Settings > Security and enable Allow HTTP Requests.
2.
Install the plugin — Save the downloaded LuauCopilot.lua to your Plugins folder: %localappdata%\Roblox\Plugins\LuauCopilot.lua
3.
Set the server URL — The plugin defaults to {{ server_url }}. Change it if running locally.
4.
Connect — Open the Agent page, copy the Session ID from the sidebar, paste it into the plugin, and click Connect.
